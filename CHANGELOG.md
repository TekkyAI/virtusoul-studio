# Changelog

## 0.1.0 (2026-02-19)

### Features

- **Chat Playground** — ChatGPT-style interface with streaming, tool calls, thinking blocks
- **Text Selection Actions** — Select text → Explain, Simplify, Deep Dive, Translate, Bookmark
- **Bookmarks** — Save and browse important messages and text selections
- **Search** — Cmd+K instant search across all conversations
- **Agent Management** — Edit SOUL.md prompts via gateway RPC
- **Config Editor** — View/edit OpenClaw config with sensitive field masking
- **Cron Manager** — Create, edit, delete, and manually trigger cron jobs
- **Session Viewer** — Monitor active sessions with token usage
- **API Middleware** — Browser never talks to OpenClaw directly
- **PostgreSQL** — Persistent chat history, bookmarks, search
- **Docker** — Multi-stage Dockerfile + Docker Compose for deployment
- **VirtuSoul Design** — Dark-first UI with lime accent theme from virtusoul.io
